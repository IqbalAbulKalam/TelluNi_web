# **App Name**: TelluNi Landing

## Core Features:

- Hero Section: Displays a captivating headline, sub-headline, logo and dual CTA buttons ('Checkout' and 'Lihat lihat dulu' (See More)) that guide the user.
- About Us: Provides a brief overview of TelluNi Fashion Store, highlighting its value proposition.
- Featured Collections Carousel: Showcases a responsive image carousel of Baju Bodo designs, featuring auto-slide with 5-second intervals, manual navigation, and breakpoints for mobile (1 slide), tablet (2 slides), and desktop (3 slides).
- Customer Gallery: Displays a responsive, optimized gallery of customer photos in a grid layout.
- Testimonials Section: Features customer review cards with ratings to build trust.
- WhatsApp Integration: Incorporates a floating WhatsApp button in the bottom-right corner, linking directly to admin support for immediate assistance.
- Image Management: Uses a simple, no-code tool that facilitates quick image updates in both the collections carousel and the customer gallery.

## Style Guidelines:

- Primary color: Gold (#D4AF37) for a luxurious and traditional feel.
- Background color: Cream (#F5F5DC), a desaturated variant of gold that provides a soft, elegant backdrop. 
- Accent color: A muted reddish-orange (#CD5C5C), a more desaturated color analogous to the primary, provides a point of contrast that suggests excitement and encourages clicks.
- Headline font: 'Playfair' serif with an elegant, fashionable feel; body text: 'PT Sans' for clear readability.
- Clean and elegant design following a mobile-first, responsive approach, ensuring seamless adaptability across all devices. It also use lazy loading on all the images
- Elegant, minimalist icons that complement the gold and cream color scheme.
- Subtle fade-in and slide-in animations on elements as the user scrolls for a smooth, engaging experience.
